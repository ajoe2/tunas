# CLAUDE.md

## Project Status
Scaffolded with `uv init`. Contains only a `main.py` entry point.

## Tooling & Commands
Managed via `uv` with Python 3.12. Do not use `pip` or bare `python` commands.

- **Run app:** `uv run main.py`
- **Add dependency:** `uv add <pkg>` (use `--dev` for development tools)
- **Sync environment:** `uv sync`
- **Run tool/script:** `uv run <cmd>`
